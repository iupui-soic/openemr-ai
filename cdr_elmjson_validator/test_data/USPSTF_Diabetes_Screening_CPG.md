# USPSTF Diabetes Screening

## Target Population

### Inclusion Criteria (ALL must be met)
1. **Age**: 35-70 years
2. **BMI**: >= 25 kg/mÂ² (overweight/obese)

**Logic**: Age AND BMI (not OR)

### Exclusion Criteria
- Existing Type 1 or Type 2 diabetes diagnosis
- Currently on diabetes medications

### Gender
- Screen ALL genders equally (no gender-based exclusion)

## Recommendation Output
When criteria met, recommend:
- Fasting plasma glucose test
- HbA1c test

## Common Implementation Errors
1. Wrong age range (e.g., 45-75 instead of 35-70)
2. Using OR instead of AND for inclusion criteria
3. Using hypertension instead of BMI as risk factor
4. Excluding patients based on gender
5. Missing BMI check
6. Not excluding existing diabetes patients
